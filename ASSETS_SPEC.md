# 《撸猫猫》小游戏美术与音效素材需求文档 (Asset Specification)

本文档列出了复刻版《撸猫猫》微信小游戏所需的所有美术与音效素材。为了确保最佳的显示效果与流畅度，请遵循以下规范。

---

## 1. 技术规范 (Technical Standards)
- **格式**：所有图片素材建议使用 **透明背景的 PNG**。
- **分辨率**：
  - 猫咪主体：建议 **512 x 512 px** 或更高。
  - UI 图标：建议 **128 x 128 px**。
  - 背景图：建议 **750 x 1334 px** (适配主流手机屏幕比例)。
- **音频**：建议使用短促的 **MP3** 或 **WAV** 格式，大小控制在 500KB 以内以加快加载。

---

## 2. 模式一：梳毛毛 (Brush Jjaemu)
*视角建议：背面、侧后方视角或俯视视角，重点展示背部。*

| 素材名称 | 文件名建议 | 状态描述 |
| :--- | :--- | :--- |
| **猫咪闲置** | `brush_idle.png` | 猫咪安静地趴着，背部平稳。 |
| **猫咪舒服** | `brush_happy.png` | 被撸时的表情，可以是眯眼、身体有轻微震动感。 |
| **预警状态** | `brush_alert.png` | 听到动静，耳朵扇动或头微微侧偏。 |
| **回头盯梢** | `brush_looking.png` | 完全转过头，正对镜头，眼神锐利。 |
| **咬人瞬间** | `brush_bite.png` | 张开大嘴扑向镜头的瞬间（Game Over）。 |

---

## 3. 模式二：摸猫猫 (Pet Jjaemu)
*视角建议：正面视角，清晰展示头部、下巴、肚子。*

| 素材名称 | 文件名建议 | 状态描述 |
| :--- | :--- | :--- |
| **猫咪正面闲置** | `pet_idle.png` | 正面蹲坐或趴着，眼神放松。 |
| **猫咪眯眼开心** | `pet_happy.png` | 摸到头或下巴时，眯起眼睛，露出笑容。 |
| **猫咪不爽警告** | `pet_annoyed.png` | 摸到肚子时，飞机耳、瞪眼或露出牙齿。 |
| **正面咬人** | `pet_bite.png` | 正面攻击的瞬间。 |

## 5. 音效需求 (Audio/SFX)

| 场景 | 音效描述 |
| :--- | :--- |
| **梳毛声** | 持续的沙沙声，可以根据滑动速度调整播放频率。 |
| **猫咪呼噜** | 开心时发出的低频呼噜声。 |
| **哈气/警告** | 猫咪生气时的“哈”声或低吼。 |
| **咬人声** | 突然的咔嚓声或衣服撕裂声。 |
